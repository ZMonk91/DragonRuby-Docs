    Making A Button#

    View Finished Source Code
    class Button
  attr_accessor :outputs, :inputs

  def DisplayButton(x, y, text)
    button_border = [x, y, 150, 150, 0, 0, 128]
    button_label = [x + 50, y + 75, text]
    outputs.borders << button_border
    outputs.labels << button_label

    if inputs.mouse.click
      if inputs.mouse.click.point.inside_rect? button_border
        outputs.labels << [200, 300, "Clicked"]
      end
    end
  end
end

def tick args
  $button = Button.new
  $button.outputs = args.outputs
  $button.inputs = args.inputs
  $button.DisplayButton(500, 400, "Click")
end

    Creating a class#

    Begin by deleting everything inside mygame/app/main.rb so we can start from a blank slate. Let's begin by creating a class called Button

    class Button
end
    This is similar to any other Object Oriented Language, where we are defining our class. Directly underneat it, with a tab space, add the attribute that allows us to get and set instance variables, with attr_accessor. Let's include the DragonRuby's outputs and inputs methods. 
    class Button
    attr_accesor :outputs, :inputs 
    end
    We can now access the outputs and inputs methods defined within the engine. These handle the I/O and Rendering.

    Defining a method#

    Let's begin creating the method for displaying our button. On a new line (inside the Button class), define the DisplayButton method. 

    class Button
attr_accesor :outputs, :inputs
    def DisplayButton
    end 
    end

    Adding Parameters#

    Let's add parameters to it. There are two ways to handle this. The first way is to write (x,y, text) and the second is to just write x y text. Both ways are valid in Ruby. For this tutorial, we will use the parenthesis approach.
    As you can see, we are not defining the types for these variables. That is because Ruby is a dynamically typed language, just like Python. The x variable will be our x position and the y will be our y position. The text variable will be our string value.
    The x and y values will automatically be inferred as an integer unless we use floating point values. 

    class Button
attr_accesor :outputs, :inputs
         def DisplayButton(x, y, text)
      end

end

    Adding variables, borders, and labels#

    Now let's assign some array values into our own variables for the border and labels. The values we are going to assign to our border will look like this:
    [x position, y position, width, height, red, green, blue]
    Let's add our own border:
    button_border = [x, y, 150, 150, 0, 0, 128]

    For the border, the values we are going to assign will look like this:
    [x position, y position, text value]
    Let's add our own label:
    button_label = [x + 50, y + 75, text]

    Your code should now look like this: 

    class Button
attr_accesor :outputs, :inputs

  def DisplayButton(x, y, text)
             button_border = [x, y, 150, 150, 0, 0, 128]
             button_label = [x + 50, y + 75, text] 
      end

end
    Now we need to assign our new array variables to the engine's output methods in order to have the engine display our new border and label.
    To do this, we need to call DragonRuby's outputs.borders and outputs.labels respectively. Then assign the array's using << ARRAY.
    Our code should look like this: 

    class Button
attr_accesor :outputs, :inputs

  def DisplayButton(x, y, text)

             outputs.labels << button_label

    Adding Click Events#

    Now for the tricky part. We need to register our click events. This will require a nested if statement because we need to check for a regular click event, then check if it happens within the border of our button.
    To do this, we need to listen for the click event from the mouse:
    if inputs.mouse.click
    Then from within that if statement, we need to check if we are clicking inside of the rectangle from an object that is of the type rect, like this:

    if inputs.mouse.click.point.inside_rect?(button_border)
    The ? denotes that that this is nullable.

    The last thing we need to do is to write the action we want to happen once our button is clicked. Within your second if statement, write:
    !#ruby outputs.labels << [200, 300, "Clicked"]
    This will generate a new label to tell us if our button was clicked. Your code should now look like this:

    Creating Instances#

    On a new line, inside of your def tick args function, create a new instance of your button using $button = Button.new
    This creates an instance of our new Button class. The $ indicates that this is a global variable.

    Now on a new line, write
    $button.outputs = args.outputs
    to specify that the attr_accessor called outputs is the same as the arg's outputs.

    Let's include the inputs as well, using
    $button.inputs = args.inputs

    Your code should now look like this:
